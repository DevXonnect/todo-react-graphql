# todo-react-graphql
TODO project made with React, Redux, GraphQL, and PostgreSQL

## Features

### Frontend
- [x] Apollo client
- [x] Redux
- [x] Load all TODO tasks
- [x] Add new TODO task
- [x] Remove TODO task
- [ ] Edit TODO task
- [ ] Mark TODO task as completed
- [ ] Sort TODO task by priority
- [ ] Sort TODO task by due date


### Backend
- [x] Node and Apollo Server
- [x] Apollo Server
- [x] Knex
